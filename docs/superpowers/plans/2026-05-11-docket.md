# Docket 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个本地运行、低负载、Vercel 风格的桌面任务管理应用（Windows + macOS），基于 Tauri v2 + Solid.js + SQLite。

**Architecture:** 四层结构——Solid.js 前端负责渲染（信号驱动状态），Tauri Commands（Rust）负责全部业务逻辑，rusqlite 直连 SQLite 本地单文件，tauri-plugin-notification + tray 后台提醒。

**Tech Stack:** Tauri v2, Solid.js + TypeScript + Vite, Tailwind CSS, SQLite (rusqlite)

**Spec:** `docs/superpowers/specs/2026-05-11-docket-design.md`

**Frontend Architecture:** `docs/superpowers/specs/2026-05-11-docket-frontend.md`

## 前置环境搭建

### 系统要求

| 依赖 | 版本要求 |
|---|---|
| Rust | 1.70+ |
| Node.js | 18+ |
| Windows | Visual Studio Build Tools (C++ workload), WebView2 Runtime |
| macOS | Xcode Command Line Tools |

### 项目初始化步骤

```bash
# 1. 创建 Tauri v2 + Solid.js 项目
npm create tauri-app@latest docket -- --template solid-ts
# 选择: SolidJS + TypeScript + Vite

# 2. 进入项目目录
cd docket

# 3. 安装前端依赖
npm install

# 4. 安装 Tailwind CSS
npm install -D tailwindcss @tailwindcss/vite
# 配置 vite.config.ts 添加 tailwindcss 插件

# 5. Rust 依赖（Cargo.toml 中的 [dependencies]）
# serde = { version = "1", features = ["derive"] }
# serde_json = "1"
# rusqlite = { version = "0.31", features = ["bundled"] }
# chrono = { version = "0.4", features = ["serde"] }
# tauri-plugin-notification = "2"
# log = "0.4"
# env_logger = "0.11"

# 6. 验证环境
npm run tauri dev
```
